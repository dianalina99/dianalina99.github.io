PORTFOLIO

Procedural Generation in 2D Video Games

I have structured my third year project (bachelor thesis) as an exploration of procedural generation techniques in game development. The project was focused on creating and documenting a 2D role-playing game using several procedural generation techniques for level generation. I have used constructive methods such as Cellular Automata, Grammars and Agent-based generation. The project report can be accessed here . The resulting game can be accessed at here , using password "dianathirdyearproject".

Particle System Simulation

This project was part of the third year "Advanced Computer Graphics" course at the University of Manchester. I have chosen to model a fire as a particle system, using C++. The open-source library of choice was SFML , which was used for generating 2D graphics.

Exploit - Unity Strategy Platformer

"Exploit" is the first 2D strategy game developed by the UniCS GameDev Society. I contributed to the first stage of development as an Unity C# Programmer. We conducted ourselves as an actual game company within a team of 5 people by creating a hierarchy or roles and building a development plan with time estimates and milestones. My role involved contributing to the initial story planning and co-developing the logic structure of the game using C# and the Unity Game Engine.
